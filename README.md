# ⚡ PCR Control

> Sistema inteligente para gestão e cronometragem de protocolos de Reanimação Cardiopulmonar (PCR).

[![Link do Projeto](https://img.shields.io/badge/Acesse_o_App-Live_Demo-10d97e?style=for-the-badge&logo=netlify)](https://pcrcontrol.netlify.app/)

O **PCR Control** é uma ferramenta técnica desenvolvida para auxiliar profissionais de saúde no monitoramento rigoroso de ciclos de RCP, administração de fármacos e registros de eventos em tempo real, garantindo que nenhum intervalo seja perdido.

---

### ✨ Destaques
* ⏱ **Precision Timer:** Contagem total do evento e ciclos automáticos de 2 minutos.
* 💊 **Drug Tracker:** Registro de Epinefrina e Amiodarona com alertas de intervalo.
* ⚡ **Defibrillation Log:** Contador de choques e controle de carga (Joules).
* 🩺 **Rhythm Monitor:** Interface para identificação rápida de ritmos (FV, TVSP, AESP, Assistolia).
* 📜 **Auto-Timeline:** Histórico detalhado gerado dinamicamente para relatórios.
* 📈 **ECG Simulator:** Visualizador de onda em tempo real via Canvas API.

### 🛠 Tecnologias
* **Interface:** HTML5 & CSS3 (Syne & JetBrains Mono Fonts)
* **Lógica:** JavaScript Vanilla (ES6+)
* **Engine Visual:** Canvas API para simulação de sinal cardíaco.
* **Deploy:** Netlify

### 🚀 Uso Rápido
1. Acesse: [pcrcontrol.netlify.app](https://pcrcontrol.netlify.app/)
2. Clique em **INICIAR PCR** para disparar o cronômetro mestre.
3. Utilize os botões laterais para registrar drogas e choques conforme o protocolo.
4. Ao final, consulte a **Linha do Tempo** para o fechamento do prontuário.
